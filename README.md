这是一个为Unity开发的高级武器后坐力系统，提供了全面的后坐力模拟和自定义功能。
该系统允许开发者创建逼真的武器后坐力效果,
支持固定模式和程序化生成的后坐力模式，并通过曲线编辑器实现精确的时间控制。
Unity版本2022.3.16
